# transformer_arch_train


## инструменты для создания примитивной ллм с архитектурой трансформер


#### датасет 
для начала возьмите часть датасета например ultrachat и поместите в папку проекта чтобы преобразовать датасет запустите remake_dataset.py или remake_dataset2.py и отредактируйте их для вашего датасета это подготовит его и создаст файл для обучения токенезатора

обучите токенезатор make_tok.py

токенезируйте данные датасета my_tokenize.py

#### тренировка модели 

задайте параметры модели и запустите train.py

для применения модели задайте в test.py идентичный класс модели как в трейн и такоеже количество слоев и запустите


архитектура трансформенра описана в transformerlayer.py
